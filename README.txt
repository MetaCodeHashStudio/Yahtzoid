 - Debug Windows 7/8
--------------------------------------------------

	1. Open Start.Bat
	2. Play the game til you crash or try to crash it.
	3. Copy the Error messages and Email to debug@metacode.com
	   with the subject: Error.crash

 - Debug Mac OSX
--------------------------------------------------
	
	1. Open up Terminal.app
	2. Type into Terminal.app:

            chmod +x 
            
	!!Do not hit return!!

	3. Drag the start.command into Terminal.app
	4. Hit return 
	5. Play the game til you crash or try to crash it.
	6. Copy the Error messages and Email to debug@metacode.com
	   with the subject: Error.crash
	

	- From this point on you can start Yahtzoid by double-clicking start.command. 

	- Warning:
		To stop Yahtzoid, do not close the terminal/command prompt window.
		Instead, type 'stop' into the console. Closing the terminal window without stopping the
		game could lead to corruption of the save files.
